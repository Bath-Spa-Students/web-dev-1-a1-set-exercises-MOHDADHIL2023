A Reference Guide For Common Of HTML Tags & CSS Properties With Some Examples

HTML TAGS

  • <h1> - Maximum Heading Tag
    <h6> - Minimum Heading Tag

    The <h1> tags shows the most important heading tag. & The <h6> tags shows the least important heading tag.

    <h1>Heading Level 1</h1>
    <h2>Heading Level 2</h2>
    <h3>Heading Level 3</h3>
    <h4>Heading Level 4</h4>
    <h5>Heading Level 5</h5>
    <h6>Heading Level 6</h6>

  • <p> - Paragraph Tag

    The <p> tags allows to make a Paragraph into the website.

    <p>Dubai is calling me, and I must go.</p>

  • <br> - Line Break Tag

    The <p> tags shows to display the next line to another line.

    habibi come to Dubai<br>
next to new line.

  • <a> - Anchor Tag

    The <a> tags defines a hyperlink Reference to another website

    <a href="https://www.w3schools.com">Visit W3School Website</a>

  • <img> - Image Tag

    The <img> tags to add some images into their webpages.

    <img src="photo.jpg" alt="Examples Of Any Photos">

  • <div> - Div Tag

    The <div> tags shows a division into the HTML documents.

    <div class="description">
        <h1>This is the first heading</h1>
        <p>Description Here</p>
</div>

  • <id> - Id Attributes

    The <id> tags attribute is used to specify a unique id for an HTML element.

    <h1 id="myDubai">My Dubai</h1>

  • <class> - Class Attributes

    The <class> tags attribute is used to specify a class for an HTML element.

    <div class="city">
        <h2>Abu Dhabi</h2>
        <p>Abu Dhabi is the capital of UAE</p>
</div>

  • <ul> - Unordered List

    The <ul> tag defines an unordered list.

    <ul>
<li>item 1</li>
<li>item 2</li> 
</ul>

  • <ol> - Ordered List

    The <ol> tag defines an ordered list.

    <ol>
<li>item 1</li>
<li>item 2</li>    
</ol>

CSS PROPERTIES

  • Background Color Property

    {background-color: orange}

    Sets the color of the background.

  • Background Image Property

    {background-image: url("photos.jpg")}

    Sets the image of the background.

  • Color Property

    {color: orange}

    Sets the color of the text.

  • Font Size Property

    {font-size: 24px}

    Sets the font size of the text.

  • Width Property

    {width: 100px}

    >Sets the width of the element.

  • Height Property

    {height: 100%}

    Sets the height of the element.

  • Padding Property

    {padding: 100px}

    Sets the distance between the border and the content.

  • Display Property

    {display: inline}

    Sets how the elemnt should be displayed.
